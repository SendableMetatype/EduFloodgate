# EduFloodgate

A [Floodgate](https://github.com/GeyserMC/Floodgate) fork that adds Minecraft Education Edition player support for online mode servers.

## About

EduFloodgate extends Floodgate to handle Education Edition players, which lack Xbox Live authentication. It provides edu-specific UUID generation, username prefixing with tenant identification, and proper data propagation for online mode servers.
